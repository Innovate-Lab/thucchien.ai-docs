# Sinh video (Bước 3/3): Tải video

## <Badge method="GET" /> `/gemini/download/v1beta/files/{video_id}:download?alt=media`

Khi video đã sẵn sàng, kết quả kiểm tra trạng thái sẽ chứa một URI. Bạn sử dụng `video_id` trích xuất từ URI này để tải video về.

<ApiReferenceLayout>
  <LeftColumn>
    ### Tham số đường dẫn (Path Parameters)
    <ApiParameter name="video_id" type="string" required isEnd>
      ID của video cần tải về, được trích xuất từ `uri` ở Bước 2. Ví dụ: `3j6svp4106e7`.
    </ApiParameter>
    
    ---

    ### Phản hồi (Response)
    Phản hồi thành công sẽ là dữ liệu video (ví dụ: file `video/mp4`).
  </LeftColumn>
  <RightColumn>
    <InteractiveApiRequest
      method="GET"
      apiUrl="https://api.thucchien.ai/gemini/download/v1beta/files/{{video_id}}:download?alt=media"

      }}

  </RightColumn>
